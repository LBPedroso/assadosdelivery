Instruções rápidas para execução local (XAMPP):
1) Copie a pasta do projeto para o diretório do servidor, por exemplo: C:\xampp\htdocs\assados-delivery
2) Importe o SQL para criar o banco de dados (arquivo principal): database/schema.sql
   - Opcional: database/tornar_email_telefone_opcionais.sql (alterações adicionais / migração)
3) Configure as credenciais do banco em: config/database.php (usuário/senha/host)
4) Acesse no navegador: http://localhost:8080/assados-delivery/ (ajuste a porta se necessário)
